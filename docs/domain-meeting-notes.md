# YIA ウェブサイト移行について — ドメイン（yia.jp）の確認事項

## この資料の目的

YIAの新しいウェブサイトが完成に近づいています。新サイトを「yia.jp」で公開するために、いくつか確認・対応が必要な点があります。この資料ではその内容をご説明します。

---

## そもそも「ドメイン」とは？

「yia.jp」は、インターネット上の「住所」のようなものです。

- ウェブサイトの住所：**yia.jp**
- ウェブサイトそのもの（建物）：今は業者が作ったサイト → 新しいサイトに入れ替えたい

ドメインの管理には以下の2つがあります：

1. **ドメインの所有権** — 「この住所は誰のものか」
2. **ドメインの設定（DNS）** — 「この住所にアクセスしたら、どの建物（サーバー）に案内するか」

今回やりたいことは、**住所（yia.jp）はそのまま**で、**案内先を新しいサイトに変更する**ことです。

---

## 現在の状況（WHOIS調査結果）

インターネット上の公開情報を調べたところ、以下のことがわかりました：

| 項目 | 内容 |
|------|------|
| ドメイン名 | yia.jp |
| 所有者（登録者） | **特定非営利活動法人 横須賀国際交流協会（YIA）** |
| DNS管理 | So-net（ソニーネットワークコミュニケーションズ） |
| 有効期限 | 2026年10月31日 |
| 最終更新日 | 2025年11月1日 |

### ポイント

- **ドメインの所有者はYIAです。** 業者のものではありません。これは非常に良い状況です。
- **有効期限は2026年10月31日です。** これを過ぎると「yia.jp」が使えなくなり、他の人に取られる可能性があります。期限前に更新（延長）が必要です。
- **最終更新は2025年11月です。** おそらく自動更新されていますが、確認が必要です。

## YIAに確認したいこと

### 質問1：So-netのアカウント情報はありますか？

ドメインの設定はSo-netで管理されています。設定を変更するには、So-netの管理画面にログインする必要があります。

**確認していただきたいもの：**
- So-netのログインID・パスワード（メールや書類に残っていないか）
- 業者から受け取ったマニュアルや書類にSo-netの情報が記載されていないか
- So-netからのメール（更新通知、請求書など）がどのメールアドレスに届いているか

### 質問2：ドメインの更新料は誰が支払っていますか？

ドメイン（yia.jp）を維持するには、年間の更新料がかかります（通常、年間数千円程度）。

**考えられるパターン：**
- **YIAが直接So-netに支払っている** → YIAが管理している可能性が高い
- **業者が支払い、YIAに請求している** → 業者経由で管理されている
- **よくわからない** → 業者に確認が必要

### 質問3：業者との契約内容はどうなっていますか？

業者との契約書があれば確認したい点：
- ドメイン（yia.jp）の管理について何か書いてあるか
- ウェブサイトの保守契約が終了した場合、ドメインはどうなるか
- 現在も保守契約は続いているか、すでに終了しているか

## 確認結果に応じた対応

### パターンA：So-netのログイン情報がわかる場合（一番簡単）

1. So-netの管理画面にログイン
2. DNS設定を変更（新しいサーバーの情報を入力）
3. 新サイトが「yia.jp」で表示されるようになる

→ **業者への連絡は不要。自分たちで対応できます。**

### パターンB：ログイン情報がわからないが、業者に連絡できる場合

業者に以下をお願いする：

> 「yia.jpのDNS設定を変更したいです。以下の設定に変更をお願いできますか？」
> - Aレコード：76.76.21.21
> - www の CNAME：cname.vercel-dns.com

→ **業者が対応してくれれば完了です。**

また、今後のことを考えて以下もお願いすると良いです：

> 「今後はYIAで直接ドメインを管理したいので、So-netの管理画面のログイン情報を教えていただけますか？」

### パターンC：業者と連絡が取れない、または対応してもらえない場合

ドメインの所有者はYIAなので、So-netに直接問い合わせることができます。

- So-netのサポートに連絡
- 「yia.jpの登録者（YIA）ですが、管理画面にアクセスできません」と伝える
- 本人確認の後、アカウント情報を再発行してもらえる可能性があります

## 更新料（有効期限）について

ドメインの有効期限は **2026年10月31日** です。

- 期限が切れると「yia.jp」でサイトにアクセスできなくなります
- 長期間放置すると、ドメインが第三者に取得される恐れがあります
- **誰が更新料を支払っているか**を確認し、今後もYIAで確実に更新できる体制を整えてください
