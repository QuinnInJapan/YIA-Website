Left: original Western-style. Right: revised for Japanese 操作マニュアル format.

❌ Before — Western Style
sections: - id: homepage title: "ホームページの編集" intro: "ホームページツールでは..." subsections: - id: homepage-hero title: "ヒーロー画像を変更する" screenshot: "homepage-hero-section" steps: - text: "左サイドバーから..." - text: "画像をクリック..." tips: - "推奨サイズ: 1920x1080" warnings: - "公開ボタンを押さないと..."
✅ After — Japanese 操作マニュアル Style
sections: - id: homepage title: "第1章 ホームページの編集" can_do: ← この章でできること - "トップページの画像を変更できます" - "お知らせの表示を管理できます" screen_overview: ← 画面の説明 screenshot: "homepage-overview" regions: - label: "①サイドバー" description: "各ツールへの..." - label: "②編集エリア" description: "コンテンツを..." - label: "③プレビュー" description: "変更の確認..." subsections: - id: homepage-hero title: "1.1 ヒーロー画像を変更する" procedure: ← 操作手順 screenshot: "homepage-hero" steps: - text: "【ホームページ】をクリック" callout: 1 ← ①赤丸 - text: "【ヒーロー画像】をクリック" callout: 2 notes: ← 補足 - "推奨サイズ: 1920×1080px以上" warnings: ← 注意 - "【公開】を押さないと反映なし" faq: ← よくある質問 - q: "画像が表示されません" a: "ファイル形式をご確認..." troubleshooting: ← 困ったときは - symptom: "画像がぼやける" solution: "1920px以上の画像を..."

New Additions Summary

+

Front Matter

はじめに (purpose) ・ 本マニュアルの使い方 (icon legend) ・ 改訂履歴 (revision table)

+

Screen Overview per Section

画面の説明 — annotated screenshot with numbered regions (①②③) explaining what each area IS before showing how to use it

+

「この章でできること」per Section

Bullet list at the top of each chapter: what you'll be able to do after reading this

+

Screenshot Callouts (①②③)

Each step can reference a numbered red circle on the screenshot. Generation script overlays these on the captured image.

+

FAQ + Troubleshooting per Subsection

よくある質問 and 困ったときは — answers common issues right where users encounter them

+

Glossary (用語集)

Back matter explaining CMS terms: 下書き, 公開, ホットスポット, スラッグ, etc.

+

【Bracket】Convention for UI Labels

Every clickable element wrapped in【】brackets — standard in Japanese software manuals
